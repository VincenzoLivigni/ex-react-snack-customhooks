Snack 1: useSwitch() – Toggle Booleano

✅ - 1 Creare un custom hook per alternare un valore booleano tra true e false.
    ✔️ - Inizialmente false.
    ✔️ - Ritorna il valore attuale e una funzione toggle() per cambiarlo.
    ✔️ - Quando toggle() viene chiamato, il valore passa da true → false e viceversa.
